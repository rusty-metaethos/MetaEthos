## Design System Principles

### Layout
- **Max width:** 1280px for content
- **Grid:** 12-column responsive grid
- **Spacing scale:** 4px base (4, 8, 16, 24, 32, 48, 64, 96, 128)
- **Whitespace:** Generous—let content breathe

### Components

**Buttons:**
- Primary: Purple gradient background, white text, subtle glow on hover
- Secondary: Transparent with purple border, purple text
- Sizes: Small (py-2 px-4), Medium (py-3 px-6), Large (py-4 px-8)

**Cards:**
- Dark slate background (#1E293B)
- Subtle border (1px, #334155)
- Hover: Lift effect + purple border glow
- Padding: 32px

**Navigation:**
- Fixed on scroll
- Backdrop blur effect
- Minimal, clean links
- Purple underline on active

**Forms:**
- Dark inputs with purple focus ring
- Floating labels
- Inline validation
- Generous padding

### Animations

**Micro-interactions:**
- Button hover: Scale 1.02, glow effect (200ms ease)
- Card hover: translateY(-4px), shadow increase (300ms ease)
- Link hover: Purple underline slide-in (150ms ease)

**Page Transitions:**
- Fade in on scroll (intersection observer)
- Stagger children animations (100ms delay)
- Smooth scroll behavior

**Loading States:**
- Skeleton screens (not spinners)
- Pulse animation on placeholders
- Progressive image loading

### Responsive Breakpoints
- Mobile: < 640px
- Tablet: 640px - 1024px
- Desktop: > 1024px
- Wide: > 1280px

## Implementation Notes

### Tech Stack
- **Framework:** React 18 + TypeScript
- **Styling:** TailwindCSS
- **Icons:** Lucide React
- **Animations:** CSS transitions + Intersection Observer
- **Forms:** Controlled components with validation
- **Build:** Vite

### Performance Targets
- **Lighthouse Score:** 95+ across all metrics
- **First Contentful Paint:** < 1.5s
- **Time to Interactive:** < 3.5s
- **Bundle Size:** < 200KB (gzipped)

### Accessibility
- WCAG 2.1 AA compliance
- Semantic HTML
- ARIA labels where needed
- Keyboard navigation
- Focus indicators
- Color contrast ratios: 4.5:1 minimum
